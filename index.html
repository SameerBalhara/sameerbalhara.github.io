Devlog 0 - Procedural Terrain Generation

Procedural terrain generation has been progressing well. We implemented chunk-based streaming with LOD meshes and began working on biome blending. River generation has gone through several iterations — first using FBM contour tracing, then Voronoi centerlines. Currently, we are exploring gradient-based carving to improve realism and continuity across chunk borders...
